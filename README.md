# personal-finance-tracker
The Personal Finance Tracker is a beginner-friendly yet practical command-line application designed to help users take control of their personal finances. It allows you to record income, log expenses, monitor savings goals, and visualize spending behaviour.
All financial data is stored securely using SQLite, while Matplotlib is used to display clear and informative charts.




View Financial Summary



Displays total income, total expenses, and the final balance.
Provides a quick snapshot of your financial status.





Savings Goal Tracking





Set a savings target and monitor how close you are to achieving it.





Visualize Spending




Choose from charts such as bar graphs, pie charts, or line plots to understand spending habits.
Useful for identifying patterns and managing finances more effectively.





Technologies Used




Python – The primary programming language used to build the application
SQLite3 – Lightweight database for storing all transaction records
Matplotlib – Library used for generating financial visualizations





How to Run the Application




1. Clone the repository
git clone <repository_url>


2. Open the project folder
Navigate to the folder named Personal Finance Tracker.


3. Run the program
python main.py





Once the program starts, you’ll see a menu with the following options:

Add Income: Enter the income amount and description.
Add Expense: Enter amount, category, and description of the expense.
View Summary: Shows totals and remaining balance.
Set Savings Goal: Define and track your financial goals.
Visualize Spending: Select a chart type to display your spending pattern.
